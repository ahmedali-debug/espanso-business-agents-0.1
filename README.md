# 🚀 Espanso AI Agent Arsenal: B2B & Venture Studio Edition

A powerful collection of 25 Espanso text-expansion triggers designed to instantly deploy highly specialized AI agents. 

This configuration bridges the gap between everyday text expansion and Agentic AI orchestration. By typing a simple shortcode (e.g., `:a_cfo` or `:a_cto`), you can instantly populate your LLM of choice with deeply contextualized system prompts tailored for complex business operations.

### 🎯 Core Focus Areas:
* **Executive Strategy & Gamification:** Agents designed to manage cognitive load, enforce time-blocking, and apply RPG-style mechanics to daily executive workflows.
* **B2B Operational Automation:** Specialized roles to streamline and automate bottlenecks across HR, legal compliance, and contract management.
* **Venture Studio Acceleration:** Pitch architects and strategy analysts engineered to map out startup synergies and craft compelling fundraising narratives.
* **Quantitative & Market Analysis:** Data-driven agents focused on institutional-grade financial modeling, fair value estimations, and competitor research.
* **Product & Tech Leadership:** CTO and PM agents tailored to guide the lifecycle of scalable, multi-agent systems and AI-driven tech products.

**Stop writing repetitive system prompts. Type a trigger, summon an expert agent, and execute.**

## 🛠️ Installation & Setup

### Step 1: Install Espanso
If you don't have Espanso installed yet, download the free, open-source application for Windows, macOS, or Linux:
👉 **[Download Espanso Here](https://espanso.org/install/)**

### Step 2: Add the Agents File
1. Open your terminal or command prompt.
2. Find where Espanso stores its configuration files by running this command:
   ```bash
   espanso path
   ```
3. Look for the `Config` path in the output. Navigate to that folder and open the `match` directory inside it.
4. Download the `agents.yml` file from this repository.
5. Place `agents.yml` directly into the `match` folder.
6. Espanso usually detects changes automatically, but to be safe, restart it by running:
   ```bash
   espanso restart
   ```

## 🚀 How to Use

Once Espanso is running in the background, open any text field—whether it's the chat bar in Gemini, a Notion document, or an email—and type one of the triggers listed below. 

The shortcode will instantly disappear and be replaced by the full, detailed system prompt, ready to guide your AI.

**Example:**
Typing `:a_rpg` will instantly output:
> *Act as an Executive Productivity & Gamification Coach. Frame your advice using RPG-style mechanics (tracking stats like XP, vitality, and cognitive load). Help me optimize my workflow using time-blocking, the 8-8-8 rule, and the Pomodoro technique to maintain a flow state.*
